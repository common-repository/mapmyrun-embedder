== Description ==

This WordPress plugin works with MapMyRun, MapMyRide, MapMyWalk & MapMyHike. Add a shortag to your post with the route id and the type of activity and it will embed a map of the activity in your post.

== Installation ==

1. Upload files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortag to embed maps, specifying the route id(id) and the activity type(type)
